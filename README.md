## 🌟 Project Overview

This project is a **Real-Time Hand Gesture Recognition System** that uses **Computer Vision + Deep Learning** to detect and classify hand gestures through a webcam.

It enables **touchless interaction**, making it useful for:

* 🖥️ Human-Computer Interaction
* 🎮 Gaming Controls
* 🧏 Sign Language Recognition
* 🤖 AI-based Automation

---

## 🎯 Key Features

✨ Real-time gesture detection
✨ Deep learning model (.h5)
✨ Simple training & testing pipeline
✨ Lightweight and easy to run
✨ Custom dataset support

```

│── Dataset/                  📁 Training images
```

## ⚙️ Installation

```bash
git clone https://github.com/Likhitha216/Hand_Gesture_project.git
cd Hand_Gesture_project
pip install opencv-python numpy tensorflow
```

---

## ▶️ How to Run

### 🔹 Train Model

```bash
python train_model.py
```

### 🔹 Run Real-Time Detection

```bash
python real_time_test.py
```
